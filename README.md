# 프로젝트 소개(AI 이용 전자책 웹사이트)
**숙명여자대학교 졸업프로젝트**로 작성한 코드입니다. 
"읽은 책을 오랫동안 기억하고 싶다"는 생각에서 출발하여, '사용자 취향을 기반으로 맞춤형 책을 추천'하고 '독서 후 AI가 던져주는 질문에 답변'하며 **독서인들의 밀도 있는 독서 생활을 지원**하는 웹사이트를 제작했습니다.

## 사용 스택 및 데이터 출처

- React
- SCSS
- Spring Boot
- git 파일 관리
- ebook 데이터: <https://www.gutenberg.org/>
